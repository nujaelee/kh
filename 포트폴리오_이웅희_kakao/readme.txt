1.  포트폴리오 소개 (카카오톡 홈페이지)
주색:흰색 검정 / 보조색:#fee500(노란색)

2. 페이지 구성
2-1 메인페이지
헤더: 상단고정

헤더탑: 	좌측: 로고(클릭시 새로고침)
	중간: 호버시 마우스 외 영역 회색 / 클릭시 하위 카테고리열림
	우측: 돋보기 아이콘 클릭시 검색영역 / 달 아이콘 클릭시 검정, 희색 반전

헤더바텀: 우측: 사용자명 / 마이페이지(클릭시 마이페이지 연결) / 로그아웃(클릭시 회원가입 / 로그인으로 바뀜)

인트로1, 2: 한 번만 드래그 해도 다음 이미지탑이 헤더의 바텀으로
          한 번 더 드래그 하면 인트로 부분 페이지에서 사라짐.

로고/텍스트박스: 기능없음.

카카오 기능 설명 영역: 좌우 버튼으로 화면넘어감.

카카오 부가 서비스 설명 영역: 스크롤 내려도 화면에 고정되다가 일정 스크롤 내려가면 다음 화면이 화면에 고정

추천 컨텐츠: 호버하면 주변 그림자, 해당 박스 살짝 위로

2-2 마이페이지
비밀번호 변경 버튼 누르면 변경할 비밀번호 입력, 확인 창 표시.
입력해서 두 값이 같은 경우에만 비밀번호 변경이 완료.

2-3 회원가입

전체: 메일주소에 입력후 다음 버튼 눌렀을 때  기존 아이디 존재하거나 문자열에 '@'와 '.' 없거나 .이 @이보다 앞에 있으면 알림창
(존재하는 ID: user0@.부터 user9@.까지)
이메일 정상 입력되면 비밀번호 입력창과 비밀번호 확인 창이 표시.
입력한 두 값이 같으면 이름 입력창 표시. 입력 후 다음 누르면 자동 종료 
하단: 로그인 클릭시 로그인 페이지로 전환


2-4 로그인

DB에 존재하는 아이디와 비밀번호가 일치하면 다음눌렀을때 화면 종료
(존재하는 ID: user0@.부터 user9@.까지 PW: pw0부터 pw9까지)
하단: 회원가입 클릭시 회원가입 페이지로 전환
